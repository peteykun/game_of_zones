Frequently Asked Questions


Compliation and Execution


When I compile my program on the Windows command prompt I get the error:
'<filename>' is not recognized as an internal or external command, operable program or batch file.

Your PATH variable is not set up properly. Please use this guide to set your PATH variable.

Default paths:

  • Java (JDK 8): C:\Program Files (x86)\Java\jdk1.8.0_40\bin
  • gcc and g++ (minGW): C:\Program Files (x86)\CodeBlocks\MinGW\bin
How do I compile my code using the terminal or command prompt?
On Windows

To compile code, you have to open command prompt in the working folder, you can do that by performing a shift + right click on a blank area inside Windows Explorer and selecting the context menu option Open Command Window Here. Alternatively, you can open the command prompt from the start menu and use cd /d <path>.

On Linux

Open a terminal and use cd <path> to change to directory or if you have the option of opening terminal in your right click context menu then select it from there.

Once you have your PATH variable set up properly, use:

  • C: gcc <filename>.c -o <filename>.exe
  • C++: g++ <filename>.c -o <filename>.exe
  • Java: javac <class_name>.java

Finally, execute your program:

  • C: ./<filename>.exe
  • C++: ./<filename>.exe
  • Java: java <ClassName>
How should I provide my program the input? How should I save the output?

While you can manually copy and paste the files provided to you, we encourage you to use I/O redirection.
To do this, simply use the following commands:

  • C: ./<filename>.exe < input.txt > output.txt
  • C++: ./<filename>.exe < input.txt > output.txt
  • Java: java <ClassName> < input.txt > output.txt

where input.txt is the filename of the file provided to you and output.txt is the desired filename of the output file.

You can then select the file output.txt (and your source file, of course) on the submission page and press submit.


Scoring and Questions


How are points calculated?

Each person will start earning points on capturing a zone. The points earned is proportional to the level of the last question solved.

Level Points earned per minute
1 10
2 20
3 30
4 40
5 50

Points are only earned for a zone when that zone is active (i.e. it is open for users to solve questions in).

When a zone reopens, the last conqueror of the zone (from the previous day) will continue to be awarded the same points (for example 50 points/minute if they were Lv. 5) until a new person attacks and solves a Lv. 1 question. Once a new Lv. 1 question is solved, the participant to solve it first gets awarded 10 points/minute, replacing the Lv. 5 conquerer from the previous day.

Are the questions different for every day of the same zone?

Questions that have been solved or revealed the previous day will be replaced by new questions.
Questions that were still locked for all users when the zone closed will not be replaced.

Are the questions different for each person attacking the same zone?

No, all users attacking the same zone get same questions.

How many zones can be active at a time?

Only one zone is active at a time. The active zone will automatically be highlighted when you open the map page.
Each zone is active for 3 hours. When a new zone is made active, we send out an email to notify all users.
